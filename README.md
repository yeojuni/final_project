## 1.프로젝트 내용
scikit learn으로 tumor dataset을 가져오고 scikitlearn이 제공하는 패키지를 통해 classification을 해보았다. 

특히, tumor데이터를 가장 높은 정확도로 분류해내는 모델을 찾기 위해 노력을 기울였다.
